A Open Source platform for models on the web

{{< blocks/link-down color="info" >}}
{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Modelix is a open source platform that aims to bring modeling on the web. Modelix goal is to allow editing models in the browser and to interact with models and services around them over web-standard APIs. Right now Modelix can act as a drop in replacement to create webbased editor for existing languages in JetBrains MPS. {{% /blocks/lead %}} {{< blocks/section color="dark" type="features">}} {{% blocks/feature icon="fa-lightbulb" title="See Modelix in action!" url="/docs/examples/" %}} Get started with our example projects and try modelix yourself. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/modelix/modelix" %}} We do a [Pull Request](https://github.com/modelix/modelix/pulls) contributions workflow on **GitHub**.
